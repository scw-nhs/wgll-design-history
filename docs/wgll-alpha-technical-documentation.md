## Discovery

The WGLL alpha prototype provides a refreshed version of the guidance, aligned with the NHS 10 Year Plan, Medium Term Planning Framework, Data Saves Lives (2022), refreshed Caldicott Principles, NHS People Plan and the Digital Capability Framework for the Workforce (2023).

The Discovery (Oct 2025) showed that while WGLL is widely used — including in social care — it lacked practical benchmarking capability and was difficult to translate into operational planning.

Key findings:
- Users want WGLL to be a practical benchmarking and improvement tool.
- Need for simplified examples and clearer measurement capabilities.
- Opportunity for WGLL to become a living, adaptive framework.

---

## Service users and needs

Primary users include CIOs, CTOs, digital transformation leads, procurement teams, IT managers, and NHSE assessment teams.

Unmet needs include:
- HR and workforce capability
- NHS App adoption
- EPR optimisation
- Primary care improvement
- Digital neighbourhoods

Setup steps included domain, hosting, SSL, installing WordPress + theme, password protection, and user roles.

## New imports

Admin → WP All Import → New import

Steps:
1. Upload spreadsheet
2. Select template
3. Map fields
4. Save template
5. Import
6. Reindex search tools

---

## Re-importing

Admin → All Import → Manage Imports

Templates can be updated without re-uploading Excel.
