## WGLL design history

This site records how the What Good Looks Like rapid prototype has iterated from October 2025 to March 2026. The Alpha service was designed, tested and iterated by a cross-functional user-centred team from [NHS SCW CSU](https://www.scwcsu.nhs.uk/).

This history is built using the [x.govuk NHS.UK plug-in for Eleventy](https://x-govuk.github.io/nhsuk-eleventy-plugin/).
